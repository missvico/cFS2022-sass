Comandos

  1. Iniciar un proyecto de node:

    npm init --yes

  2. Instalar sass de forma global

    npm install -g sass

  3. Agrego los scripts para compliar sass en mi package.json

    "scripts": { "sass-watch-dev": "sass --watch -c --style=expanded --no-source-map styles/sass:styles/css", "sass-compile-dev": "sass -c --style=expanded --no-source-map styles/sass:styles/css", "sass-compile-prod": "sass -c --style=compressed styles/sass:styles/css" }
